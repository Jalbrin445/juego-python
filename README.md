# Proyecto Final de Semestre

## Descripción

El presente proyecto se trata de un juego de validación de contraseñas generadas por el programa de forma aleatoria con el fin de aplicar los principios básicos de la programación orientada a objetos utilizando el lenguaje de programación Python, y los principios de aleatoridad, validación estrictia y manejo de errores, donde cada estudiante tendrá un rol de cazador y el programa será el encargado de otorgarle las contraseñas de forma aleatoria y a su vez, validar que todo ha funcionado correctamente, ahora bien, este proyecto hace parte del curso de programación ofertado de forma electiva en el pregrado de Ingeniería Electrónica en la Universidad Nacional Abierta y A Distancia (UNAD), por lo tanto, no es un proyecto grande sino de aprendizaje.

## Dependencias

Para este proyecto se utilizaron dos librerías, las cuales son:

* random (para generar los valores aleatorios)

* string (para extraer datos de constantes con caracteres predefinidos)
